一、Caddyfile

Caddy的配置文件使用Caddyfile，当caddy启动时会默认查找当前路径下的Caddyfile作为配置启动

二、配置内容(基本)

Site Addresses

类似于nginx的listen和server_name相结合的一条配置，指定了监听的域名和端口，有趣的是当你指定域名并且使用默认端口时，caddy会默认打开http和https功能并且不需要自己配置证书。

Path Matching

相当于nginx的location关键字，用以匹配url执行响应的操作。
